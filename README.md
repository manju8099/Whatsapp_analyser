# WhatsApp Analyser 🤖

A Node.js bot that automatically analyzes WhatsApp messages and sends scheduled email reports with a built-in dashboard.

## Features ✨
- 📊 Automatic WhatsApp message analysis
- 📧 Scheduled email reports via Nodemailer
- 📈 Built-in analytics dashboard
- ⏰ Task scheduler for automated reports
- 💾 Local message storage and tracking

## Tech Stack 🛠️
- **Runtime:** Node.js
- **WhatsApp:** whatsapp-web.js
- **Email:** Nodemailer
- **Scheduler:** Node-cron
- **Validation:** Zod
- **Frontend:** HTML/CSS/JavaScript

## Installation ⚙️

1. Clone the repository
   git clone https://github.com/manju8099/Whatsapp_analyser.git

2. Install dependencies
   npm install

3. Create a .env file and add your credentials
   EMAIL=your_email@gmail.com
   EMAIL_PASSWORD=your_app_password

4. Run the bot
   node bot.js

## Usage 🚀
- Scan the QR code with WhatsApp to connect
- Bot will automatically start analyzing messages
- View analytics on the dashboard
- Receive scheduled email reports
